ESP-VISION 编程指南
===================

:link_to_translation:`en:[English]`

ESP-VISION 是面向乐鑫 ESP32-P4 与 ESP32-S3 开发板的 MicroPython 视觉运行时。
它整合了 MicroPython ESP32 移植、OpenMV ``imlib`` 图像处理算法以及 ESP-DL 模型推理，
让你可以使用 Python 构建嵌入式视觉应用。

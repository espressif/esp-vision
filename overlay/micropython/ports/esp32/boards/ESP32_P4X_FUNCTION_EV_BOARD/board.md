The following firmware is applicable to the ESP32-P4 Function EV Board based on
ESP32-P4.

This board must be equipped with at least 16 MiB external SPI Flash and PSRAM.
The default configuration enables the ESP32-P4 target, the ESP32-P4 Function EV
Board flash layout, external ESP32-C6 hosted WiFi/BLE wiring, and the SC2336
MIPI-CSI camera baseline used by ESP-VISION.
